## 🎨 Customization Guide

### Update Pricing (Easiest!)
Edit `src/data/services.js` - Change any price instantly

### Add New Service
Add object to services array - Component renders automatically

### Change Colors
Search for `bg-blue-600`, `text-blue-600` - Replace with any Tailwind color

### Update Company Info
Edit Navigation.jsx and Footer.jsx with your details

### Connect Form to Backend
Update Contact.jsx with your API endpoint
